- Rendering Videos on 2D & 3D Objects
- Configurable Aspect Ratio
- Portrait / Landscape Orientation 
- Adaptive Bitrate(ABR) and 8K (UHD) Video

- Synchronization of video with audio
- AES-128 encryption
